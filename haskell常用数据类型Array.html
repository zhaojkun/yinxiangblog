haskell常用数据类型Array

haskell常用数据类型Array
初始化：
import Data.Array
let a = array (0,10) [(i,i) | i<-[0..10]]
let b = listArray (0,10) [0..10]
let c = accumArray (+) 0 (1,3) [(1,2),(1,4),(2,1),(2,2),(2,3),(3,5)]

查询内容：

let firstElem = a ! 0   // 查询某一个数据，0表示序号
let bs = bounds a     // bounds 返回数组的边界
let idxs = indices a   // indices 返回所有的序号
let elemss = elems a   // elems 返回所有的元素
let relations = assocs a  // assocs 返回（序号，数据）的元组序列

更新内容：

let au = a // [(0,1),(1,3)]  更新内容，update的数据是一个元组序列
